## [ShadowCat: File Transfer via Animated QR Codes Using Fountain Codes](https://github.com/unprovable/ShadowCat) ⭐️ 7.0/10

ShadowCat is a new open-source browser-based tool that transfers files between devices by encoding data into a stream of animated QR codes using fountain codes, allowing reliable recovery even if some codes are missed. This approach enables file transfer without any network connection, making it useful for air-gapped devices or when traditional communication is broken, such as a phone with a damaged radio. The tool runs entirely in a browser as a single-page application, chunks the file, encodes chunks into QR codes displayed in sequence, and the receiving device uses a camera to capture and reassemble the file with CRC verification.

hackernews · unprovable · May 22, 11:11 · [Discussion](https://news.ycombinator.com/item?id=48234287)

**Background**: Fountain codes are rateless erasure codes that can generate an unlimited number of encoding symbols from source data, allowing recovery from any subset of received symbols. Animated QR transfer streams QR codes in rapid succession, and fountain codes ensure that even if some codes are not captured, the file can still be reconstructed.
