This technique could provide a new therapeutic avenue for cancers that lack effective targeted drugs, potentially expanding the reach of precision medicine to many more patients. Unlike previous Cas9-based approaches that only damage DNA at the target site, Cas12a2 shreds the entire chromatin once activated, making it far more destructive. ## [Adaptive PDFs: Embed Markdown in PDF via ZIP](https://sgaud.com/texts/pdf) ⭐️ 7.0/10

A technique called Adaptive PDFs embeds markdown source files inside a PDF by creating a ZIP archive with the PDF as the first file, making the file both a valid PDF and a valid ZIP archive. This enables dual readability: humans view the PDF normally, while LLMs or text extractors can extract structured markdown from the same file, potentially improving AI-assisted document processing and reducing format friction. The trick requires the PDF to be the first file in the ZIP archive and uses compression level 0 to ensure the PDF remains intact; a simple script can automate the process.

hackernews · SarthakGaud · Jun 12, 16:32 · [Discussion](https://news.ycombinator.com/item?id=48506209)

**Background**: A polyglot file is a file that is valid in multiple formats simultaneously, such as a PDF-ZIP polyglot. PDF readers ignore trailing data after the end-of-file marker, while ZIP readers look for the central directory at the end of the file, allowing a single file to be interpreted as both formats.

In 2007, an engineer discovered that shouting at hard disk drives in a data center causes measurable performance degradation due to acoustic vibration. This finding highlights the sensitivity of HDDs to environmental noise, which remains relevant for data center design and storage reliability, especially as HDD capacities increase. The engineer observed that shouting at HDDs caused a noticeable drop in throughput, likely due to acoustic resonance interfering with the read/write head positioning.

rss · PC Gamer · Jun 12, 11:17

**Background**: Hard disk drives rely on precise mechanical movements of a read/write head over spinning platters. Acoustic vibrations, especially at resonant frequencies, can disturb this positioning, causing the drive to retry operations and thus reducing performance. This phenomenon has been studied further in the context of acoustic denial-of-service attacks.
