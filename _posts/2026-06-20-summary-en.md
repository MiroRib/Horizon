## [AMD to Reinstate TSME on Ryzen 9000 via July BIOS Update](https://www.tomshardware.com/pc-components/cpus/amd-will-reinstate-memory-encryption-on-ryzen-9000-cpus-through-a-bios-update-in-july-tsme-is-coming-back-after-valuable-community-feedback) ⭐️ 6.0/10

AMD announced it will restore Transparent Secure Memory Encryption (TSME) on Ryzen 9000 desktop processors through a BIOS update in July, reversing a previous silent removal of the feature. This reversal shows AMD responding to community backlash, preserving a security feature that protects against physical memory attacks, which is valuable for users concerned about data theft from stolen or compromised systems. TSME encrypts all data in system memory to prevent cold boot and physical access attacks. The feature was removed via AGESA firmware update on non-PRO Ryzen 9000 CPUs, but will be reinstated after community feedback.

hackernews · roboror · Jun 20, 19:19 · [Discussion](https://news.ycombinator.com/item?id=48612098)

**Background**: Transparent Secure Memory Encryption (TSME) is a hardware-based memory encryption technology that encrypts the entire system memory transparently to the OS and applications. It is distinct from AMD's SME (Secure Memory Encryption) which is OS-controlled. TSME is typically enabled on Ryzen PRO processors for enterprise security, but was also available on some consumer Ryzen 9000 CPUs via BIOS option.
